# Robot-Diferencia_Parra_Octavio
PRÁCTICA INTEGRADORA: ROBOT DIFERENCIAL, Cinemática directa, cinemática inversa y control cinemático, Robótica Móvil by OCTAVIO PARRA

Instrucciones:
Dentro de las carpetas "Cinemática directa", "Cinemática inversa" y "Control", se encuentran subcarpetas denominadas "código".
El código se encuentra en formato de texto .txt, para correr estos códigos, se debe copiar y pegar el texto completo en Matlab -Se recomienda versión R2025b en adelante- después de pegar el texto, correr el código, y se observaran las gráficas y animaciones.
